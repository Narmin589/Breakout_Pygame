# Breakout Game

Classic Breakout-style brick breaking game built with Python and Pygame.

## Features
- Break all bricks to win
- 3 lives system — lose a life when the ball drops
- Built using Object-Oriented Programming (OOP)
- Sprite-based game objects (ball, paddle, bricks)
- Collision detection system

## How to run
1. Install dependency: `pip install pygame`
2. Run: `python Breakout_game.py`

## Technologies used
- Python
- Pygame
- OOP (classes and objects)
- Sprite management
- Collision detection
